# 1. What is the difference between var, let and const?
## ans: var is function-scoped and can be redeclared, which often leads to bugs, whereas let and const are block-scoped for better predictability. While let allows you to reassign values later, const creates a constant reference that cannot be reassigned after its initial declaration.

# 3. What is the difference between map(), filter(), and forEach()?
## ans: forEach() executes a function for each element without returning a new array, while map() creates a new array by transforming every element. filter() also returns a new array, but only includes elements that meet a specific logical condition.

# 5. What are template literals?
## ans: Template literals are a modern way to handle strings in JavaScript, defined by backticks (`) instead of quotes. They allow for easy string interpolation and multi-line strings by using the ${expression} syntax to embed variables directly into the text.
